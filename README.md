# MineFleet Booking

MineFleet Booking adalah aplikasi web untuk mengelola pemesanan kendaraan operasional pada perusahaan tambang nikel. Aplikasi ini mendukung proses pemesanan kendaraan, penentuan driver, approval berjenjang, monitoring pemakaian kendaraan, konsumsi BBM, jadwal service, riwayat pemakaian, dan export laporan ke Excel.
